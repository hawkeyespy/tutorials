# Java Lambda, Streams, and Functional Programming Chapters

1. Lambda expressions and functional interfaces
2. Method references and constructor references
3. Stream basics: source, filter, map, and forEach
4. Stream reductions: count, sum, min, max, reduce, and collect
5. Working with Optional
6. Collectors: grouping, partitioning, joining, and mapping
7. Primitive streams and performance-friendly stream choices
8. Functional style in real code: composition, immutability, and side effects

Each chapter has three files:

- `lessons/ChapterXX...java` explains the topic with runnable examples.
- `practice/ChapterXX...Practice.java` gives TODO exercises.
- `solutions/ChapterXX...Solution.java` shows one clean solution.

Run examples from the repository root:

```powershell
javac com\rc\functional\lessons\Chapter01LambdaExpressions.java
java com.rc.functional.lessons.Chapter01LambdaExpressions
```
